#Shopping List
This is an example application based on React, NodeJS and MongoDB


## Technologies
React
NodeJS
MongoDB

## Intallation
```console
npm install
```

## Testing
```console
npm install -g mocha
npm test
```

If you run Mocha from IntelliJ add the following to "Extra Mocha options":
```
--compilers js:babel-core/register --require ../../../test/test_helper.js
```
